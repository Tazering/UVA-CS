### **Mergesort**
```python
MERGE-SORT(list, first, last)
    if first < last
        mid = (first + last) / 2
        MERGE-SORT(list, first, mid)
        MERGE-SORT(list, mid+1, last)
        MERGE(list, first, mid, last)
    return
```
